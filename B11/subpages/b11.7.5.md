Normally multipart monographs that are not analyzed/classified as a collection (e.g., one
bibliographic record is made for all volumes) are not in scope for MLC.  However, if a book cataloged
as MLC because it seems to be a single-part monograph, then a subsequent volume is received that lacks
an analyzable title, change the existing bibliographic record for the first volume to a collected set record
and add information about the new volume following DCM C12.  Because MLC resources are stored
off-site, the holdings and item records cannot be update by the cataloger.  Forward the added volume to
Kay Guiles in PSD with an explanation of what needs to be done.
