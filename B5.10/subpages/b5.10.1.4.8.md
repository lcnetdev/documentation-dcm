Treat as a translation.  Record up to six codes for the languages of the
text, each in a repeated $a subfield following the guidelines in DCM B5.10.1.2.  If more than six
languages are involved, input code “mul” (multilingual).  Record in subfield $h the code for the
language of the original.
( **NOTE:** Prior to November 1990, only the code for the predominant text (or
code “mul,” if no language predominated or if there was doubt that one did) was
input in subfield $a.)
EXAMPLE 1: Text in English and French, translated from Vietnamese.
Input:  041 1# $a eng $a fre $h vie
Pre-Nov. 1990 practice: 041 1# $a mulvie
*(Note code “mul”)*
EXAMPLE 2: Text in English and French; originally published in Inuit, with English
translation.
Input:  041 1# $a eng $a fre $h esk
Old practice:  041 1# $a mulesk
*5.10.1.4.9 Mixture of translations and originals*
If a resource contains separate pieces in various languages, some of
***Government Publication Codes***
which are translations and some of which are originals, follow DCM B5.10.1.2 to record languages of the
text.  Do **not** treat as a translation.
