At the IBC stage, records will continue to be identified as being "in-process" with encoding
level 5 (this is because certain internal and external data manipulations are done off this value).
Once an item enters the cataloging stream, the workflow is the same as for other levels of
cataloging with the following exceptions: staff doing the initial aspect

1. set Leader/17 to 4;
2. add an 042 field containing "pcc".
The code "pcc" identifies records prepared by members of the Program for Cooperative
Cataloging; it is also used in LC core level records to call attention to their "coreness." LC
original cataloging other than core level will not carry this identification.
