The language of cataloging is the language used in the physical description, notes, etc.  Do
not confuse this with the language of text (008/35-37; 041 $a subfield).  Subfield $b has been
defined to contain a code for the language used in cataloging.  If subfield $b is not present in a
record, assume the language of cataloging to be English.  With respect to copy cataloging,
observe the following two points:

1. external records are candidates for use in copy cataloging **only** if the language of
cataloging is English (040 $b not present or, if present, contains “eng”);
