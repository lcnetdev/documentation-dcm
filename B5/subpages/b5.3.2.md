Most cataloging at LC is done using the Multiple Use MARC System (MUMS) for
input/update. All cataloging of new serials and maintenance to existing serial records is done on
the OCLC system (copies of the records are transmitted to LC via tape and loaded into the
SERIALS file). All new cataloging of items in the JACKPHY languages (Japanese, Arabic,
Chinese, Korean, Persian, Hebrew, Yiddish) is done on the RLIN (Research Libraries
Information Network) system, since that system has the capacity to include nonroman as well as
roman scripts. Copies of the records are transmitted to LC via tape for distribution. Preparations
are under way also to load the records into a read-only file so that they will be available for
searching and display (roman data only). Records in JACKPHY languages input at LC prior to
the availability of use of RLIN were input in romanized form via the MUMS system. See DCM
B5.11 (Appendix 6) for the guidelines used to determine what is cataloged via the RLIN system.
