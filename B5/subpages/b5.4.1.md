That aspect of cataloging called "descriptive cataloging" consists, in general, of determining
the non-subject access points, including determining the responsibility of authorship/corporate
emanation by selecting a "main entry," doing the authority work to support the accessing system
for names, providing the bibliographic description of the item, and performing any catalog
maintenance stimulated by the cataloging of a particular item. The steps to accomplish this may
be summarized as follows:

1. determine the choice and form of main entry heading and all other access points;
2. search the LC Database to determine if needed headings are established:
a. if found in NAMES file in AACR 2 form (and so coded), use the heading (unless
other change is needed);
b. if found in NAMES file but not coded for AACR 2, code for AACR 2, making all
other needed changes including evaluating references;
c. if not found in NAMES file, establish as new heading, using information in the LC
Database, if any;
d. if heading already established in non-AACR 2 form, consider any needed
record/catalog maintenance as specified in DCM C13;
e. perform other record/catalog maintenance determined to be needed that is discovered
during performing the preceding steps.
3. complete bibliographic description; assign appropriate fixed field values.
