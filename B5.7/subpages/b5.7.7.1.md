The person doing the subject aspect records/checks data on the STARS Slip as follows:
*Statistics area:* Records in **arabic numerals** the number of subject authorities
made/changed; the number of class numbers established/changed; the number of bibliographic
records changed.
*Call number, etc:* Records the call number for the shelflister (the descriptive cataloger
analytic in a collected set in the blank area below the collected set call number.
