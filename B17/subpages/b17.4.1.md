In general, give added entries in the order stated in LCRI 21.29, giving edition-specific
added entries first followed by copy-specific added entries. These guidelines are illustrated as
follows:
7XX (700-740) General added entries; within this group, give any added entries with
relators (subfield $e) in the order stated in LCRI 21.29, preceding any name/title
added entries in those categories in which name/title added
entries are appropriate; give copy-specific added entries in the same
order, following any edition-specific added entries of the same type:
